ARCHIVO PRINCIPAL: test-colab.ipynb
PLATAFORMA: Google Colab (100% gratuito con GPU T4)

================================================================================
PASOS PARA EMPEZAR (5 MINUTOS)
================================================================================

1. SUBIR NOTEBOOK A COLAB

   Ir a: https://colab.research.google.com
   File > Upload notebook
   Seleccionar: test-colab.ipynb

2. ACTIVAR GPU

   Runtime > Change runtime type
   Hardware accelerator: GPU
   Save

3. OBTENER API KEY

   Ir a: https://app.roboflow.com
   Settings > API Keys
   Copiar "Private API Key"

4. EJECUTAR NOTEBOOK

   Ejecutar celdas en orden (Shift+Enter):

   Celda 1: Instalar dependencias (2-3 min)
   Celda 2: Verificar GPU Tesla T4
   Celda 3: Importar modulos
   Celda 4: Definir clase JerseyAnalyzer
   Celda 5: Definir interfaz Gradio
   Celda 6: PEGAR API KEY y ejecutar

5. USAR INTERFAZ

   Se generara URL publica: https://xxxxx.gradio.live
   Abrir en navegador
   Subir imagen de camiseta
   Click "Analizar"
   Ver resultados
